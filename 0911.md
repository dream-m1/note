* 逻辑运算符
AND 与   两个结果一样为真  
OR  或   两个结果一个真则为真  
NOT 取反 取反操作  
XOR 异或 两个不一样结果为真  
NOR 同或 两个结果相同为真  
> 一个数XOR 1 为相反数，XOR 0 为本身 ，XOR 本身 为 0 

* 移位操作  
  - 左移：往左移动后面填0，左移一位相当整个数乘以2 
```
0000 1111 << 0001 1110 << 0011 1100
```
  - 右移：往左移动前面填0，右移一位相当整个数除以2
```
0000 1111 >> 0000 0111 >> 0000 0011
```
