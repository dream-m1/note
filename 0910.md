## 进制转换

* 基数 ：基于某个进制衍生的数，2进制基于2，最大的数字比基数小一；每个数字都要乘以基数次幂

1. 任意进制转十进制  
    - (1001.01)<sub>2</sub>＝1×2<sup>3</sup> + 0×2<sup/>2</sup> + 0×2<sup>1</sup> + 1×2<sup>0</sup> + 0×2<sup>-1</sup> + 1×2<sup>-2</sup>＝ 9.25   
   - (25)<sub>8</sub>＝2×8<sup>1</sup> + 5×8<sup>0</sup> ＝ 21
2. 十进制转二进制  
算法：【除2 倒序 取余】 
```
    2 |_10_ -- 0
    2 |_5_  -- 1
    2 |_2_  -- 0
        1
    结果：1010
>>>>>>>>>>>>>>>>>>>>>
    8 |_10_ -- 2
         1
    结果：12
```
